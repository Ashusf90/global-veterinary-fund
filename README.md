## Client Description/Mission:
Global Veterinary Fund (GVF) focuses on providing financial support to veterinary students, 
animal/environment-focused organizations, and individuals facing veterinary care debt, while engaging 
with the community through meaningful, student-led projects and events. 

## Project Proposal:
- Basic Overview
- Home Page
- About Page (E-Board)
- Whoever is in primary schools
- Social media links
- Application Forms
- For aid, to fill out information about themselves (support to veterinary care debt, veterinary students, two separate forms)
- Submitting organization data for research (separate form)
- Embedded Google Forms
- Events page, ability to add to list
- Donation page 
- Admin panels (and login) to update forms, and events

## Technologies + Context
